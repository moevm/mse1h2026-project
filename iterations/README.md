# Отчёты по итерациям (iterations)

В этой папке находятся подпапки для каждой итерации проекта (iteration-1, iteration-2 и т.д.).  
В каждой подпапке хранятся:

- `presentation.pdf` - презентация по итогам итерации.
- `demo.mp4` - скринкаст с демонстрацией фич (кроме первой итерации).
- `tasks_planned.md` - запланированные задачи на итерацию.
- `tasks_completed.md` - выполненные задачи.
- `next_tasks.md` - задачи на следующую итерацию.

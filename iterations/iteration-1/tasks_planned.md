# План работ на первую итерацию

## Бэкенд
- Инициализация проекта NestJS, настройка TypeScript, линтера и форматтера.
- Разработка OpenAPI спецификации для основных сущностей (курсы, проекты, пользователи).
- Реализация мок-данных и заглушек API для параллельной разработки фронтенда.

## Фронтенд
- Настройка роутинга и базового компонента с проверкой работоспособности.
- Настройка линтера и форматера.

## База данных

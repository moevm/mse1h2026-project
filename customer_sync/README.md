# Встречи с заказчиком

В этой папке хранятся протоколы встреч с заказчиком.  
Каждый протокол оформляется в отдельном файле с именем по шаблону:  
`customer_YYYY-MM-DD.md` (например, `customer_2026-02-20.md`).

## Шаблон протокола
```markdown
# Созвони №X
- **Дата:** dd.mm
- **Тема встречи:** ...
- **План встречи:**
  - ...
- **Участники:**
  - ...
- **Запись встречи:** url
- **Результаты и договорённости:**
  - ...

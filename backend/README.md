# mse-backend

## Установить зависимости

```bash
pnpm i
```

## Используемые в Backend технологии

- `TypeScript` - строго типизированный язык на основе JavaScript, расширяющий его возможности;

- `NestJS` - NodeJS фреймворк для создания серверных приложений с использованием TypeScript;

- `Prisma` - TypeScript ORM для связи backend и базы данных.

## Запустить сервер

```bash
pnpm run start
```

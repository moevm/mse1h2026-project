# Командные синхронизации

В этой папке хранятся протоколы внутренних встреч команды.  
Каждый протокол оформляется в отдельном файле с именем по шаблону:  
`Созвон №X (dd.mm).md`
## Шаблон протокола
Основные разделы протокола:
- Дата, тема, план встречи
- Участники
- Обсуждённые вопросы
- Принятые решения
- Протоколист
- Запись встречи

# mse-frontend

## Установить зависимости

```bash
pnpm i
```

## Используемые технологии

- `TypeScript` — строгая типизация JavaScript;

- `Vue 3` — фреймворк для построения интерфейсов;

- `Naive UI` — библиотека компонентов с поддержкой тем и TypeScript;

- `Vue Router` — навигация между страницами;

## Запуск в режиме разработки

```bash
pnpm dev
```

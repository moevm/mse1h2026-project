# Документация проекта (docs)

В этой папке хранятся основные документы проекта, описывающие его концепцию, требования и руководства.
- `project_overview.md` – общее описание проекта, постановка задачи, функциональные и нефункциональные требования.
- `use-cases.png` – диаграмма вариантов использования.
- `ER-diagram.png` – ER-диаграмма базы данных.
- `openapi.yaml` – спецификация OpenAPI.
